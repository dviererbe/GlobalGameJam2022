Name:
  LPC Grave Markers Rework
Version:
  1.3
Description:
  Some LPC grave markers merged into a single set & reworked
  for use in Stendhal.
Tileset details:
  - Designed for use with 32x32 tiled maps.
  - Orientation: orthogonal
  - Shadowless PNG images use indexed color (smaller files).
  - Shadowed images & GIMP sources (.xcf) use RGB color.
